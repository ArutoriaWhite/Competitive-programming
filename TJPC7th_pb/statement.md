# TJPC 7th 練習賽 pB

# 魔力測量
## 題目背景
## 題目敘述
有很多人排隊測量魔力，每個人身上的魔力值經過水晶測量後會顯示出測量結果。
若某人的原始魔力為 x ，則水晶顯示的測量結果為：$((x+10) \times 2)^2$
給你排隊的人的數量，請你依序幫每個人測量魔力。
但由於測量魔力的水晶非常脆弱，只要**測量出的結果*超過這顆水晶的最大承受魔力就會爆掉，並且接下來都不能測量了。輸入會告訴你這顆水晶的最大承受魔力。

## 輸入說明
輸入兩個正整數 $N$、$K$，分別代表$N$個人排隊要測量魔力，以及這顆水晶最大承受的魔力為$K$。
接下來一行有 $N$ 個正整數，以空白分隔，依序代表每個人的原始魔力。

## 輸出說明
請你依序對每個人的原始魔力輸出測量結果。若該人測量出的魔力使水晶爆掉了，請輸出`"BURST!!!"`(不含引號)，並且對於接下來排隊的人都不用輸出。
注意輸出後請換行。

## 子任務
- $1\leq N\ \leq 10^5$
- subtask 1: score:30 constraints: 所有原始魔力小於 $10^4$ 且 $1 \leq K \leq 10^8$
- subtask 2: score:70 constraints: 所有原始魔力小於 $10^{8}$ 且 $1 \leq K \leq 10^{16}$

## 範例輸入
```
8 77777
89 93 1 3 4 5 1000 3
```
## 範例輸出
```
39204
42436
484
676
784
900
BURST!!!
```
